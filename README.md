## 思路

- 思考
  - 项目背景是怎样的？遇到的问题？

  - 已有的解决方案是什么？为什么不行？

  - 最终带来什么样的价值？
  - 从中学习到了什么？有哪些收获？
  - 如果再来一次，有没有更好的实现方式？有哪些地方是可以优化的？
