# 源码调试

## 初始化项目
```bash
pnpm create vite
# 输入项目名称，如 study-vite，进入项目目录
cd study-vite
# 安装依赖
pnpm install
```

## 增加 debug 命令
在`package.json`中的`scripts`字段增加`debug`命令，如下：
```json
{
  "scripts": {
    // ...
    "debug": "node --inspect-brk ./node_modules/vite/bin/vite.js",
    // ...
  },
}
```

## 开始打包
```bash
pnpm debug
```
